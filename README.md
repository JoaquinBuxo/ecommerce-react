# Mobile Ecommerce

## Getting Started

### Prerequisites

- npm

### Installation

1. Clone the repository:

   ```sh
     git clone git@github.com:JoaquinBuxo/ecommerce-react.git
   ```

2. Install dependencies:

- ecommerce : `cd ecommerce-react`

  ```sh
  npm install
  ```

3. Setup and start the app:

- Client:

  ```sh
  npm start
  ```
